schematose
===
_schematose_ is a modern C++ template library generating code for checking conformance of XML data
to some schema in absence of a "real" schema validator library.

Supported C++ Standards are:
- [ ] 11
- [ ] 14
- [ ] 17
- [x] 20

Tested with: 
- [x] GCC
- [ ] clang
- [ ] MSVC
